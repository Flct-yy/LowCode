本项目采用 Monorepo 架构，使用 Yarn Workspaces 进行包管理，将多个相关项目集中在同一代码库中管理。

```
项目根目录结构：
├── apps/                  # 应用程序目录
│   ├── api/              # 后端 API 服务
│   ├── web/              # 主前端应用
│   └── preIframe/        # iframe 预览服务
├── packages/             # 共享包目录
│   ├── components/       # 共享组件库
│   ├── config/           # 共享配置
│   └── utils/            # 共享工具函数
├── docs/                 # 项目文档
├── package.json          # 根项目配置
├── tsconfig.base.json    # 基础 TypeScript 配置
└── yarn.lock            # Yarn 依赖锁定文件
```
